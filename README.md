# Perfil profesional · GitHub Pages

Web estática lista para publicar. Antes de subirla, reemplaza:

- `hola@tu-dominio.com` en `index.html` por tu email profesional.
- `https://tu-usuario.github.io/tu-repositorio/` en `robots.txt` y `sitemap.xml` por la URL final.
- El nombre `Marta` si quieres usar marca o nombre completo.

## Publicarla en GitHub Pages

1. Crea un repositorio en GitHub y sube estos archivos a la rama `main`.
2. En el repositorio: **Settings → Pages**.
3. En **Build and deployment**, selecciona **Deploy from a branch**, rama `main` y carpeta `/(root)`.
4. Guarda. GitHub mostrará la URL pública en esa pantalla al terminar el despliegue.

Para una imagen social al compartir el enlace, añade después una imagen `og-image.jpg` de 1200 × 630 px y las etiquetas `og:image` y `twitter:image` en el `<head>`.
